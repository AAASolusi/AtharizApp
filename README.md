# Maintenance Jobcard Portal

This is a maintenance jobcard portal application for managing tasks, customers, and technicians.

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables based on `.env.example`.
4. Run the development server: `npm run dev`

## Firebase
The project is configured to use Firebase for database and authentication.
- Ensure `firebase-applet-config.json` is present and correctly configured.
- Firestore rules are managed in `firestore.rules`.
